# XSTAMPP-MODEL-PARSER

This repository contains the parser for the XSTAMM model files.

Now it can parse `*.hazx` files and LTLs in it.

## build pypi package

```cmd
pip install autoflake black isort pytest
pytest -s
pip install build
python3 -m build
```
